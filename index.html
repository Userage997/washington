Epysium

Совместный проект с Hunter Voronov. Платформа для продвинутых OSINT-расследований, позволяющая автоматизировать сбор и анализ данных из открытых источников. Инструмент значительно ускоряет процесс расследований и повышает точность аналитики.

Автоматическая деанонимизация по цифровым следам
Построение связей между сущностями в реальном времени
Детектирование инфраструктуры для сноса
Визуализация сложных сетевых структур
